# Student Expense Tracker (Python)

## Overview
A simple Python program to add, view, delete and analyze student expenses.  
Runs in Google Colab or local computer.

## Features
- Add Expense
- View Expenses
- Delete Expense
- Total Spending
- Category-wise Summary
- Highest & Lowest Expense

## How to Run (Google Colab)
1. Create files using %%writefile
2. Run: !python main.py

## Technologies Used
- Python
- CSV file for storage

## File Structure
- main.py
- expense_manager.py
- analyzer.py
- data.csv
